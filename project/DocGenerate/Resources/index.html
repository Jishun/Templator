CodeExample

This page you are looking at is generated by Templator, please refer to the template as an example, correspoding calling code is here
To use this libary:
  • Initialize an instance of Templator Config. cutomize it if necessary
  • Assign an instance of ILogger to the config object, or leave it default an later use ((TemplatorLogger)config.TemplatorLogger).Errors to retrieve parser errors
  • Initialize an instans of TemplatorParser with the config object
  • Construct a IDictionary<string, object> as input or use TemplatorUtil.ParseJsonDict(string) method to parse a Json object as input
  • Load text/xml templat as string or stream
  • var outPut = parser.LoadXmlTemplate(stream, input); or parser.ParseXml, or parser.LoadTextTemplate etc.. will generate the parsed string
  • Check the logger object (((TemplatorLogger)config.TemplatorLogger).Errors by default) for parsing errors
  • parser.Holders will give out the holders defined inside the template
Helper tools:
  • Build phase validation task for msbuild/vs is ready on new get, search for 'TemplatorSyntaxBuildTask' and install it to your project which contains the template. it will check syntax for you.
  • Vs syntax highlighing extension is ready for download. click to download, install, and enjoy ^_^
